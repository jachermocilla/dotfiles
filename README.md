# vim-utils
My VIM utilities which basically is a vimrc :)

### Install 

    ./setup.sh

### Mappings
*   F8 - Invokes ctag in the current directory
*   F4 - Toggles tags list
*   F3 - Focus tags list
*   F7 - Inserts license text at the top
